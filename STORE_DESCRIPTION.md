### Cards Dictionary Helper

**Easily build your vocabulary with Cards Dictionary Helper!**

### How It Works

1. **Get Your API Token**
   - Visit kotcat.com/profile
   - Create an API token
   - Copy it to the extension settings

2. **Select Your Folder**
   - The extension loads your available folders
   - Choose where to save new words

3. **Start Learning**
   - Browse any website
   - Select a word you want to learn
   - Right-click ‚Üí "Add to dictionary"
   - Done! The word is saved with its source URL
